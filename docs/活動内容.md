# 活動内容

経験者も初心者もどなたでも大歓迎！大学から始めた人も多くいます。

## 創作活動

ボーカロイドを使用した作曲活動及びイラスト作成を行っています。
作曲、イラスト、動画編集を分担した合作の作成、ボカコレというボカロ楽曲投稿祭への参加などを行っています。

サークルメンバーと絵チャットやGartic Phoneで遊ぶことも！
創作活動を大学から始めた人も多くいるので、経験者初心者問わず大歓迎です！

## DJ活動

ボーカロイド楽曲（合成音声を使用した楽曲）をメインにDJをしています。
一橋祭でのパフォーマンスやボーカロイドクラブイベント（ボカクラ）での出演、主催をしています。

- 主催イベント → #多摩ロイド
- 所属団体 → #crossnotes

学生団体 #crossnotes では、ジャンルを問わずDJイベントの出演や参加、他大との交流などが可能です。
興味のある方はぜひDJ班にお越しください！

## ファン活動

カラオケ会やご飯会などを通して、サークル員の親睦を深めています。
昨年度は新歓期間のカラオケ会、忘年会や新年会などを行いました！

サークル員同士でライブやコラボイベントに参戦したり、お気に入りの楽曲を布教する布教会も行っています。
マジカルミライやボーマス、各地のボカクラやニコニコ超会議などに参戦！

オタク仲間が欲しい方、ボカロのイベントに行きたい方、どなたでも大歓迎です。

## 一橋祭

一橋祭ではオリジナルドリンクの提供や謎解き企画などの実施に加え、DJイベント「#ボカバシ」を開催しました。
サークル員によるDJプレイの他、豪華なゲストDJをお呼びして大盛況のイベントになりました！
